Simple news site
++++++++++++++++

Description
===========

An application written for Django1.4 using Python2.7.

Consists of two pages:
  * *news/* - displays updates sorted by date of creation
  * *news/form* - fill in this form to add new updates. Linked from the main page. 

Example templates included in the templates/ subdirectory. Basic admin
interface available.
